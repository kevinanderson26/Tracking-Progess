CB0500A1: County Business Patterns for the U.S., State, County
and Metropolitan Areas: 2005
UNIVERSE DESCRIPTION
The universe of this file is all operating establishments with
one or more paid employees. This universe includes all
establishments classified in the North American Industry
Classification System (NAICS) Codes 11 through 813990.
DATA ITEMS
This file contains data on:

  Number of establishments
  Annual payroll
  Total employment

GEOGRAPHIC COVERAGE
The data are shown at the U.S. level and by State, county, and
Metropolitan and Micropolitan Statistical Areas.
INDUSTRY DETAIL
The data are shown at the 2- through 6-digit NAICS code levels
for all sectors.
OTHER VARIABLES IDENTIFYING RECORDS
None
RELEASE SCHEDULE
The data in this file correspond to the County Business Patterns
reports that will be released in June 2007.
RELATED DATA FILES
None.
CORRESPONDENCE TO PDF REPORTS
The U.S.- and State-level data in this file are exactly the same
as are shown in the PDF-format County Business Patterns reports.
These reports are available on the Census Bureau's Internet site at
www.census.gov. The County- and Metropolitan Area data in this file
are only available in database format.
DATA LIMITATIONS
There are no special limitations to these data.
FILE SEQUENCE
Data are presented in ascending Geography by NAICS Code.
